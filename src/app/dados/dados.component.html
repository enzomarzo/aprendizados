A terceira coluna recebe o nome de ´tratamento´. E eu quero saber quais são os campos diferentes (ou unicos) que aparecem nessa coluna. Para isso é só digitar dados['tratamento'].unique(). Se eu quiser fazer a contagem de elementos é só digitar .value_counts()

Essas funções do pandas podem receber parametros. Então se eu fizer dados['tratamento'].value_counts(normalize = True) eu vou receber a relação proporcional dessa contagem de dados.

Na hora de analisar dados é interessante que exista algum tipo de criptografia dos dados, seja para não sermos tendenciosos na analise seja para proteger a informação do usuário.

dados['composto'].value_counts()[0:5]. É simples, mas temos que olhar os códigos por partes. É como ler uma frase, lendo palavra por palavra. Então temos a tabela dados e estamos pegando apenas a coluna de composto. Depois estamos contando quanto existe de cada composto. E por fim estamos pegando apenas os itens 0, 1, 2, 3 e 4. Ou seja, os 5 primeiros itens do array. Já que o value_counts lista primeiro o componente com mais frequencia, então teremos o top 5 mais frequentes.

Machine Learning

Popularizado nas duas ultimas décadas mas estudado desde a decada de 50, o Machine Learnin é o processo do computador aprender algo sem ele ter sido programado exatamente para isso (escrito por Arthur Samuel em 1959 - 'Field of study that gives computers the ability to learn without being explicitly programmed)'.

O Machine Learning se popularizou por 2 motivos principais. O primeiro é o grande volume de dados. E o segundo é o poder computacional para processar esses dados. Image por exemplo a abertura de conta em um banco. O processo de identificação da nossa imagem é um processo de machine learning. O computador consegue reconhecer quem sou eu a partir de uma analise de milhões de fotos.

Scikit Learn: é uma biblioteca bastante famosa de machine learning.
