https://alura.com/frontend?id=1

  • https:// => Protocolo que fará a comunicação com o servidor
  • alura.com => dominio que será transformado em IP e porta de entrada. Por exemplo 127.221.108.20:8080.
  • frontend => Aqui já estamos dentro do servidor. Esse seria o caminho/path que queremos acessar.
  • ?id=1 => Esse seriam os parâmetros. São as query strings.
