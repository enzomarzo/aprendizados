Porque Surgiu?

O Javascript, que nasceu como LiveScript e mudou de nome pegando carona no marketing do Java, surgiu com o intuito inicial de validar formulários HTML. Antes do JS as páginas web eram feitas apenas com HTML. O JS trouxe a dinâmica, deu mais vida aos sites. Isso era 1995. O primeiro navegador web (o Mosaic) tinha surgido apenas 2 anos antes.
