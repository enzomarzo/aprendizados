Firebase

O Firebase é um back-end as a Service. Na imersão de React que eu fiz do NLW foi utilizado o Firebase para não precisar programar o back-end. O Firebase, que foi desenvolvido pelo Google, nos concede um monte de coisas de fácil acesso.!

A exemplo, 3 funcionalidades muito úteis: Servidor + Banco de Dados + Autentição de usuário

No React ou Angular nós temos primeiro que instalar o firebase no projeto com yarn add firebase ou npm i --save firebase. Depois criamos um arquivo firebase.ts e importamos os modulos que queremos utilizar no projeto (por exemplo o /auth e o /database). Nesse arquivo da aula do NLW tem um exemplo.

Também subimos um servidor nesse projeto. É só digitar npm install -g firebase-tools , depois precisamos nos logar no CLI do firebase firebase login e depois iniciarmos com firebase init. Tudo isso tem na página do Firebase. Ao fazer o init temos que sinalizar o que queremos (por exemplo: realtime database e hosting )

O firebase vai continuar perguntando algumas coisas. O site do create-react-app tem essas informações para fazer o deploy utilizando o firebase.

depois é só dar um npm run build e firebase deploy
